## 문제

[문제 링크](https://boj.kr/2442)


<p>첫째 줄에는 별&nbsp;1개, 둘째 줄에는 별&nbsp;3개, ..., N번째 줄에는 별 2×N-1개를 찍는 문제</p>

<p>별은 가운데를 기준으로 대칭이어야 한다.</p>



## 입력


<p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>



## 출력


<p>첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.</p>
