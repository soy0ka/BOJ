[문제 링크](https://boj.kr/15700)

## 입력

<p>첫째 줄에 N과 M이 주어진다. (1 ≤ N, M ≤ 1,000,000,000)</p>

## 출력

<p>첫째&nbsp;줄에 채울 수 있는 타일 개수의 최댓값을 출력한다.</p>
