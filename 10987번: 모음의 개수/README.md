## 문제

[문제 링크](https://boj.kr/10987)

<p>알파벳 소문자로만 이루어진 단어가 주어진다. 이때, 모음(a, e, i, o, u)의 개수를 출력하는 프로그램을 작성하시오.</p>

## 입력

<p>첫째 줄에 단어가 주어진다. 단어의 길이는 1보다 크거나 같고, 100보다 작거나 같으며, 알파벳 소문자로만 이루어져 있다.</p>

## 출력

<p>첫째 줄에 모음의 개수를 출력한다.</p>
